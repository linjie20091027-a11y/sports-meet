## 🌟 緣起

澳門濠江中學，立校於一九三二年，歷九十餘載風雨，育英才數萬。校園坐落青洲之濱，紅牆綠樹，鐘聲琅琅。每逢金秋時節，運動場上旌旗獵獵，吶喊震天——這便是全校師生翹首以盼的年度盛事：**田徑運動會**。

然而籌辦一場運動會談何容易？數十個比賽項目、數百名參賽學生、繁複的報名審核、精確的賽程編排、即時的成績統計……傳統紙筆作業不僅耗時費力，更易出錯。為此，我們打造了這套**全流程數位化管理系統**，讓運動會籌備變得井然有序、高效透明。

---

## ✨ 核心功能

### 🏠 首頁大屏

開幕倒計時以秒為單位跳動，營造緊張期待的賽前氛圍。賽事總覽、最新公告、最新成績三欄並列，資訊一目了然。右側邊欄設有「報名須知」與快捷入口，新訪客亦能迅速上手。

### 🏃 賽事項目

涵蓋田徑運動會全部常規項目：短跑（100米、200米、400米）、長跑（800米、1500米）、田賽（跳遠、跳高、實心球）、接力（4×100米）及集體項目（拔河、廣播體操）。每個項目皆可設定性別組別（男子 / 女子 / 混合）、人數上限、比賽規則與場地分配。

### 📝 在線報名

學生登錄後可瀏覽所有開放項目，查看剩餘名額，一鍵提交報名。系統自動檢測重複報名、超出限報數量、性別組別不匹配等問題。報名提交後進入管理員審核流程，審核結果通過站內通知即時推送。

### 📅 賽程編排

支援自動編排與手動調整雙模式。自動編排根據項目人數、場地可用時間、參賽者衝突自動生成賽程表；手動模式允許拖拽調整場次時間與場地。一鍵發布後，學生可在個人中心查看自己的參賽時間與檢錄安排。

### 🏆 成績管理

成績錄入支援手動輸入與 Excel 批量導入。系統根據成績自動排名，支援並列名次。獎項設定靈活，可自定義一/二/三等獎及優秀獎。審核通過後一鍵公示，同步更新獲獎榜單。

### 📊 數據統計

班級排名、年級對比、參賽率、獲獎分布等多維度統計。可視化圖表（柱狀圖、餅圖）直觀展示數據。所有統計報表支援 Excel / CSV 導出與線上打印。

### 📢 公告通知

分類公告涵蓋賽事通知、報名截止、成績公示、緊急通知。置頂功能確保重要訊息不被淹沒。學生端已讀標記，管理端可查看閱讀量統計。

### 👤 用戶管理

支援批量導入學生賬號（Excel），自動生成統一格式郵箱與默認密碼。角色分為管理員與學生兩級，權限清晰。自定義頭像上傳（支援文件選擇與剪貼板粘貼）。操作日誌全程留痕，可追溯。

### 🗣️ 交流論壇

師生交流專區，支援發帖、回覆、按瀏覽量排序。學生回覆需經管理員審核後公開顯示，確保內容健康有序。帖子支援管理員刪除與通知推送。

### 🤖 AI 智能助手「小濠」

基於 DeepSeek V4 大語言模型，注入濠江中學完整知識庫（建校歷史、校訓、師資、設施、歷屆運動會佳績等）及實時運動會數據。學生可隨時向「小濠」提問，獲取準確、親切的回覆。支援粵語口語風格，如同與學長學姐對話。

### 🔒 安全機制

登錄驗證碼防機器人，JWT Token 認證，bcrypt 密碼加密存儲。失敗次數超限自動鎖定賬號。防 SQL 注入（參數化查詢）、防 XSS 攻擊（內容轉義）。操作日誌記錄所有敏感行為。

## 🎨 設計理念

> **「紅牆映日，金線描邊。書卷氣與競技魂在此交融。」**

整體採用端莊大氣的紅色主調（`#A80000`），靈感源自濠江中學校徽與北大官網風格。深灰輔色沉穩內斂，金色點綴彰顯榮耀。佈局採用不對稱雙欄結構——左主區展示核心內容，右側欄提供輔助入口。字體選用微軟雅黑與系統黑體，標題加粗強化層次。直角邊框取代圓角，硬朗線條呼應競技精神。

## 📂 項目結構

```
sports-meet/
├── server.js                # Express 服務入口
├── database/init.js         # SQLite 初始化 + 16條學校信息 + 20個默認項目
├── middleware/auth.js       # JWT 認證中間件
├── routes/
│   ├── auth.js              # 登錄/註冊/驗證碼/密碼重置
│   ├── public.js            # 公共API（賽事查詢/成績公示/公告）
│   ├── admin.js             # 管理員API（53個端點）
│   ├── student.js           # 學生API（報名/個人中心/通知/頭像）
│   └── forum.js             # 論壇API + AI助手路由
└── public/
    ├── index.html           # SPA 主頁面
    ├── css/style.css        # 全局樣式（響應式設計）
    └── js/
        ├── api.js           # API 封裝層
        ├── app.js           # 路由/導航/彈窗/Toast
        ├── auth.js          # 登錄註冊交互
        ├── admin.js         # 管理後台（10個功能標籤）
        ├── student.js       # 學生個人中心（8個功能標籤）
        └── forum.js         # 論壇前端 + AI聊天UI
```
